# Stored DOM XSS

Lab ini menunjukkan sebuah kerentanan **stored DOM** pada fitur komentar blog. Untuk menyelesaikan lab ini, eksploitasi kerentanan tersebut untuk memanggil fungsi **`alert()`**.


## Solusi

Kirim komentar yang berisi vector berikut:

```html 
<><img src=1 onerror=alert(1)>
```

Dalam upaya mencegah XSS, website menggunakan fungsi JavaScript **`replace()`** untuk melakukan encode pada tanda kurung sudut (**angle brackets**). Namun, ketika argumen pertama berupa string, fungsi tersebut hanya mengganti kemunculan pertama.

Kita mengeksploitasi kerentanan ini dengan cukup menambahkan satu set tambahan tanda kurung sudut di awal komentar. Tanda kurung sudut tersebut akan di-encode, tetapi tanda kurung sudut setelahnya tidak akan terpengaruh, sehingga memungkinkan kita melewati filter dan menyisipkan HTML.
