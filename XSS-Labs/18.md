# Reflected XSS into a template literal with angle brackets, single, double quotes, backslash and backticks Unicode-escaped

Lab ini berisi kerentanan **reflected cross-site scripting (XSS)** pada fitur pencarian blog.

Refleksi terjadi di dalam sebuah **template string**, di mana tanda kurung sudut (`< >`), tanda kutip tunggal (`'`), dan tanda kutip ganda (`"`) sudah di-encode sebagai HTML, serta backtick juga telah di-escape.

Untuk menyelesaikan lab ini, lakukan serangan cross-site scripting yang dapat memanggil fungsi `alert()` di dalam template string tersebut.


## Solusi

1. Kirimkan string alfanumerik acak ke kotak pencarian, contohnya `TESTING123`.

2. Amati bahwa string acak tersebut direfleksikan di dalam sebuah **template string JavaScript**.

```html
<script>
var message = `0 search results for 'TESTING123'`; document.getElementById('searchMessage').innerText = message;
</script>
```

3. Ganti input kamu dengan payload berikut untuk mengeksekusi JavaScript di dalam template string:

```
${alert(1)}
```

4. Verifikasi bahwa teknik tersebut berhasil dengan cara klik kanan, pilih **“Copy URL”**, lalu tempelkan URL tersebut di browser. Saat halaman dimuat, seharusnya akan memicu alert.
