# Reflected XSS with some SVG markup allowed

Lab ini memiliki kerentanan **reflected XSS** yang sederhana. Situs ini memblokir tag-tag HTML yang umum, tetapi melewatkan beberapa **tag SVG** dan event terkait.

Untuk menyelesaikan lab ini, lakukan serangan **cross-site scripting (XSS)** yang memanggil fungsi **`alert()`**.


## Solusi

1. Masukkan payload XSS standar seperti:

```html id="x1q9kd"
<img src=1 onerror=alert(1)>
```

Perhatikan bahwa payload ini diblokir. Pada langkah berikutnya, gunakan **Burp Intruder** untuk menguji tag dan atribut mana yang diblokir.

2. Buka browser Burp dan gunakan fitur pencarian di lab. Kirim request tersebut ke **Burp Intruder**.

3. Di request template, ganti nilai search term menjadi:

```html id="a8p2vf"
<>
```

4. Letakkan kursor di antara tanda kurung sudut lalu klik **Add §** untuk membuat posisi payload. Nilai search term sekarang menjadi:

```html id="m4k7zr"
<§§>
```

5. Buka **XSS cheat sheet** lalu klik **Copy tags to clipboard**.

6. Di Burp Intruder (panel Payloads), klik **Paste** untuk memasukkan daftar tag ke payload list, lalu klik **Start attack**.

7. Setelah selesai, periksa hasilnya. Semua payload menghasilkan response **400**, kecuali tag:

* `<svg>`
* `<animatetransform>`
* `<title>`
* `<image>`

yang menghasilkan response **200**.

8. Kembali ke Intruder dan ubah search term menjadi:

```html id="v6n0cx"
<svg><animatetransform%20=1>
```

9. Letakkan kursor sebelum tanda `=` lalu klik **Add §** untuk membuat posisi payload. Nilai sekarang menjadi:

```html id="t3z9qp"
<svg><animatetransform%20§§=1>
```

10. Buka **XSS cheat sheet** lalu klik **Copy events to clipboard**.

11. Di Burp Intruder, klik **Clear** untuk menghapus payload lama, lalu **Paste** untuk memasukkan attribute list. Klik **Start attack**.

12. Setelah selesai, periksa hasilnya. Semua payload menghasilkan response **400**, kecuali payload **onbegin** yang menghasilkan response **200**.

13. Buka URL berikut di browser untuk memastikan `alert()` berjalan dan lab selesai:

```text id="y7k1mp"
https://YOUR-LAB-ID.web-security-academy.net/?search=%22%3E%3Csvg%3E%3Canimatetransform%20onbegin=alert(1)%3E
```
