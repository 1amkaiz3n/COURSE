# LAB API Testing

Panduan penyelesaian lab **API Testing** di PortSwigger Academy, mencakup berbagai skenario seperti insecure API endpoints, server-side parameter pollution, mass assignment, dan broken access control untuk memahami bagaimana input pengguna diproses oleh backend API.

👉 Fokus utama: mengidentifikasi endpoint API yang terekspos, menganalisis cara server memproses parameter (query/body/path), lalu mengeksploitasi kelemahan seperti manipulasi parameter, over-permission API fields, dan logika backend yang tidak tervalidasi untuk mendapatkan aksi yang tidak semestinya (misalnya akses data sensitif atau melakukan tindakan admin).
