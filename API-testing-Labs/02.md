# Finding and exploiting an unused API endpoint

Untuk menyelesaikan lab ini, eksploitasi endpoint API tersembunyi untuk membeli **Lightweight l33t Leather Jacket**. Kamu bisa login ke akunmu sendiri menggunakan kredensial berikut: `wiener:peter`.


## Solusi

1. Di browser Burp, akses lab dan klik salah satu produk.

2. Di **Proxy > HTTP history**, perhatikan request API untuk produk tersebut. Contoh:
   `/api/products/3/price`

3. Klik kanan request API tersebut lalu pilih **Send to Repeater**.

4. Di tab **Repeater**, ubah metode HTTP dari `GET` menjadi `OPTIONS`, lalu kirim request. Perhatikan bahwa respons menunjukkan metode `GET` dan `PATCH` diizinkan.

5. Ubah metode request dari `GET` menjadi `PATCH`, lalu kirim request. Perhatikan muncul pesan **Unauthorized**. Ini mengindikasikan kamu perlu autentikasi untuk mengubah order.

6. Di browser Burp, login ke aplikasi menggunakan kredensial `wiener:peter`.

7. Klik produk **Lightweight "l33t" Leather Jacket**.

8. Di **Proxy > HTTP history**, klik kanan request `/api/products/1/price` untuk jaket kulit tersebut lalu pilih **Send to Repeater**.

9. Di tab **Repeater**, ubah metode request dari `GET` menjadi `PATCH`, lalu kirim request. Perhatikan muncul error karena **Content-Type tidak sesuai**. Pesan error menunjukkan bahwa **Content-Type harus application/json**.

10. Tambahkan header `Content-Type` dan set nilainya menjadi `application/json` : `Content-Type: application/json`

11. Tambahkan body JSON kosong:

```
{}
```

lalu kirim request. Perhatikan muncul error karena body tidak memiliki parameter **price**.

```json
{"type":"ClientError","code":400,"error":"'price' parameter missing in body"}
```

12. Tambahkan parameter **price** dengan nilai 0:

```
{"price": 0}
```

13. Di browser Burp, reload halaman produk jaket kulit tersebut. Perhatikan bahwa harga sudah berubah menjadi **$0.00**.

14. Tambahkan jaket kulit ke keranjang.

15. Buka keranjang dan klik **Place order** untuk menyelesaikan lab.
