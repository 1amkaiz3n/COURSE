## 📝 9. Bug Report (Bug Bounty Format)

### Title:

> DOM-based Cross-Site Scripting (XSS) via `location.search` in search functionality

---

### Summary:

Vulnerability allows attacker to inject malicious JavaScript via URL parameter which is executed in victim's browser due to unsafe usage of `innerHTML`.

---

### Steps to Reproduce:

1. Open vulnerable page
2. Visit:

```
https://target.com/?search=<img src=x onerror=alert(1)>
```

3. Observe alert execution

---

### Impact:

* Arbitrary JavaScript execution
* Session theft (if cookies not HttpOnly)
* UI redirection / phishing
* Full client-side compromise

---

### Root Cause:

Unsafe usage of:

```js
innerHTML = location.search
```

---

### Recommendation:

* Use `textContent` instead of `innerHTML`
* Sanitize input using DOMPurify
* Encode output properly
