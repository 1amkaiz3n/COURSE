## 📌 Tabel: Escape vs Parse di DOM XSS

| Kondisi / Context                                    | Apa yang terjadi                            | Contoh Input                   | Hasil                                      |
| ---------------------------------------------------- | ------------------------------------------- | ------------------------------ | ------------------------------------------ |
| **Text context (aman / innerText / textContent)**    | Semua karakter dianggap teks biasa          | `<img onerror=1>`              | Ditampilkan literal, tidak dieksekusi      |
| **HTML entity encoding (escape karakter spesial)**   | Karakter seperti `<` `>` diubah jadi entity | `">`                           | Menjadi `&gt;` (tidak keluar konteks)      |
| **innerHTML tanpa filter (raw HTML parsing)**        | Input diperlakukan sebagai HTML             | `<b>Hello</b>`                 | Bold text                                  |
| **innerHTML + tag berbahaya**                        | HTML diparse + event dieksekusi             | `<img src=1 onerror=alert(1)>` | JavaScript jalan                           |
| **Attribute context (value="...")**                  | Input berada di dalam atribut HTML          | `" onmouseover=alert(1)`       | Bisa keluar atribut kalau tidak disanitasi |
| **Sanitizer aktif (DOMPurify / filter server-side)** | Tag/script dihapus atau diubah              | `<img onerror=1>`              | Dihapus / dinonaktifkan                    |
| **URL encoding context**                             | Input di-encode jadi URL safe               | `>` → `%3E`                    | Tidak dieksekusi, hanya data               |

---

## 📌 Intinya gampangnya

* **Escape** → diperlakukan sebagai teks biasa (`&lt; &gt;`)
* **Parse** → dianggap HTML beneran (tag jalan, event bisa aktif)

---

## 📌 Rule cepat buat kamu di lab

* Kalau keluar jadi `&lt; &gt;` → ❌ sudah di-escape (aman / dibatasi)
* Kalau muncul jadi tag HTML (`<img> <b>`) → ⚠️ diparse (potensi XSS)
* Kalau event jalan (`onerror`, `onload`) → 🔥 DOM XSS berhasil
