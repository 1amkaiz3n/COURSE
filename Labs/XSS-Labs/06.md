# Reflected XSS into a JavaScript string with angle brackets HTML encoded

Lab ini berisi kerentanan **reflected cross-site scripting (XSS)** pada fitur **pelacakan query pencarian**, di mana tanda kurung sudut (**angle brackets**: `<` dan `>`) di-encode. Refleksi input terjadi di dalam sebuah **string JavaScript**. Untuk menyelesaikan lab ini, lakukan serangan cross-site scripting yang keluar dari string JavaScript tersebut dan memanggil fungsi **`alert`**.

## Solusi

1. Masukkan string alfanumerik acak ke kotak pencarian,contohnya `TESTING123`.

2. Perhatikan bahwa string acak tersebut direfleksikan di dalam sebuah **string JavaScript**.

```html
<script>
var searchTerms = 'TESTING123';
document.write('<img src="/resources/images/tracker.gif?searchTerms='+encodeURIComponent(searchTerms)+'">');
</script>
```

3. Ganti input kamu dengan payload berikut untuk keluar dari string JavaScript dan menyisipkan `alert`:

```javascript
'-alert(1)-'
```

4. Verifikasi bahwa teknik tersebut berhasil dengan klik kanan, pilih **"Copy URL"**, lalu tempel URL tersebut di browser. Saat halaman dimuat, seharusnya akan memicu **alert**.
